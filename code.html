                            // one option to determine if fizz, buzz or fizzbuzz

                            function fizzBuzzToo(fizzValue, buzzValue) {
                                let returnArray = [];
                                // loop from 1 to 100
                                for (let i = 1; i <= 100; i++) {

                                    if(i % fizzValue == 0 && i % buzzValue == 0) {
                                        returnArray.push(`FizzBuzz`);
                                    }else if (i % buzzValue == 0) {
                                        returnArray.push(`Buzz`);
                                    }else if(i % fizzValue == 0){
                                        returnArray.push(`Fizz`);   
                                    }else{
                                        returnArray.push(i);
                                    }     
                                }
                                return returnArray;
                            }

                            // my preferred option to determine if fizz, buzz or fizzbuzz
                            function fizzBuzz(fizzValue, buzzValue) {
                                let returnArray = [];
                                // loop from 1 to 100
                                for (let i = 1; i <= 100; i++) {

                                    let value = ((i % fizzValue == 0 ? `Fizz` : '') + (i % buzzValue == 0 ? 'Buzz' : '') || i ); 
                                    returnArray.push(value); 
                                }
                                return returnArray;
                            }

FizzBuzz

FizzBuzz is an app that displays numbers from 1 to 100. For multiples of "3", instead of the number displays "Fizz" and for multiples of "5" instead of the number displays "Buzz". If the number is a multiple of both "3" AND "5" then "FizzBuzz" is displayed. This version allows the user to choose their own numbers instead of just 3 and 5.

The code is structured in four functions. Actually only three are used, but I wanted to show two different versions of figuring out the logic portion.

The first function getValues() receives the inputs from the form and calls the fizzBuzz() and resultsFizz() functions.

The next two functions accomplish the same result but I waanted to show two ways of doing it. The first fizzBuzzToo() uses a for/loop to go through the numbers and add them to an array based on whether they are divisible by the inputs from the first function. The second function fizzBuzz() uses the ternary operator to do the same thing but with much less code.

The last function resultsFizz() loops through the array and builds the cells to populate the table, adds the class to color the cell based on result and displays it on the page.
